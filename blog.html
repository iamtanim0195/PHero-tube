Discuss the scope of var, let, and const

var: Function-scoped. Hoisted to the top of the function or global scope. let: Block-scoped (inside curly braces). Not hoisted; only accessible after declaration. const: Block-scoped. Requires initialization with a fixed value; cannot be reassigned (for primitive types), but can be used to modify object properties or array elements.

Tell us the use cases of null and undefined

undefined: Default value for uninitialized variables. Represents missing function arguments or missing object properties. null: Used to explicitly indicate the absence of a meaningful value. Often used as a placeholder or to reset values in objects or arrays.

What do you mean by REST API?

REST API stands for Representational State Transfer Application Programming Interface. It is an architectural style for designing networked applications. REST is not a technology itself but rather a set of constraints and principles that can be applied to web services and web applications to make them more scalable, maintainable, and easily accessible over the internet.
